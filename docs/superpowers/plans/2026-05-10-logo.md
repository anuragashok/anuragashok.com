# Logo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a small typographic logo (Newsreader `a` + amber period) to the site header and as the favicon.

**Architecture:** Single new server component returning inline SVG; one className edit in `site-header.tsx` to wire it next to a responsively-hidden wordmark; one static SVG file under `src/app/icon.svg` so Next.js's metadata loader auto-emits the favicon link tag.

**Tech Stack:** React server components + inline SVG, Tailwind v4 (uses existing `--font-serif`, `--accent` tokens), Next.js 16 metadata file convention (`app/icon.svg`).

**Source-of-truth spec:** `docs/superpowers/specs/2026-05-10-logo-design.md`

---

## Task 1 — Logo component

**Files:**
- Create: `src/components/logo.tsx`

- [ ] **Step 1: Create `src/components/logo.tsx`**

```tsx
type Props = { size?: number; className?: string };

export function Logo({ size = 24, className }: Props) {
  return (
    <svg
      width={size}
      height={size}
      viewBox="0 0 32 32"
      role="img"
      aria-label="Anurag Ashok"
      className={className}
    >
      <text
        x="2"
        y="24"
        fontFamily="var(--font-serif)"
        fontWeight="500"
        fontSize="24"
        fill="currentColor"
      >
        a
      </text>
      <circle cx="22" cy="23" r="2" fill="var(--accent)" />
    </svg>
  );
}
```

(Coordinates `text x=2 y=24` and `circle cx=22 cy=23 r=2` were chosen so the period sits visually next to the `a`'s right edge at the baseline. If after a visual smoke check the period looks off, adjust `circle cx` (move horizontally) or `r` (size). Keep `viewBox="0 0 32 32"` so all sizes scale uniformly.)

- [ ] **Step 2: Verify typecheck + build**

Run: `pnpm typecheck && pnpm build`
Expected: green. The new component isn't imported anywhere yet so no build effect.

- [ ] **Step 3: Commit**

```bash

```

## Task 3 — Favicon

**Files:**
- Create: `src/app/icon.svg`

- [ ] **Step 1: Create `src/app/icon.svg`**

```xml
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32">
  <text x="2" y="24" font-family="ui-serif, Georgia, 'Times New Roman', serif" font-weight="500" font-size="24" fill="#0a0a0a">a</text>

```

(Coordinates match Task 1's component for visual consistency. Foreground `#0a0a0a` and amber `#f59e0b` are hardcoded because browsers don't resolve CSS variables in favicon SVGs.)

```bash
# kill the bg dev server
```

Expected: a `rel="icon" href="/icon..."` token in the output.
